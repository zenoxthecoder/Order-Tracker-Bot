# Order Tracker Bot

**Order Tracker Bot** is a powerful and easy-to-use Discord bot designed to help servers manage and track customer orders efficiently. It allows staff members to create and monitor orders, update their status, and keep everything organized inside Discord.

This bot is especially useful for servers that sell digital products, services, or commissions and need a simple way to manage orders without external tools.

With **Order Tracker Bot**, staff can quickly create order records, track their progress, and keep customers updated about their order status. The bot helps reduce confusion by keeping all order information structured and easy to access.

Features

📦 Create and manage customer orders

📊 Track order progress and status

🔔 Organized order information inside Discord

⚡ Simple and easy-to-use commands

🧾 Designed for service stores, digital shops, and commissions

Command List

!create
Creates a new order entry. Staff or Admins can use this command to register a new customer order and start tracking it.

Example Usage
!create

This command will create a new order entry that can be used to track the progress of a customer's request.

# Edit these things
Change `BASE_URL` and add your own hosting domain or cloudflare domain.

Change `PORT` and add your own hosting port or cloudflare port.

Change `TOKEN` and add your discord bot token.

# How to find token?
Go to [Discord Developer Portal](https://discord.com/developers/applications) and create a New Application.

Or watch the [tutorial](https://youtu.be/aHG1eOeZSJA) here.
